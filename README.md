Overview

The 2D Graphics Editor is a console-based graphics system developed in C programming language.
It simulates basic drawing operations using a 2D character array canvas, where shapes are represented using * and empty space using _.

This project demonstrates fundamental concepts of data structures, modular programming, and graphics simulation in C.

🎯 Objective

To design a simple graphics editor that allows users to:

Add geometric shapes to a 2D canvas
Delete shapes from the canvas
Modify existing shapes
Display the final graphical output
🧩 Features
➕ Add Shapes (Circle, Rectangle, Line, Triangle)
❌ Delete Shapes from canvas
✏️ Modify existing shapes
🖥️ Display canvas in terminal
📦 2D array-based graphics representation
🧠 Menu-driven user interface
